# Tour Agency

Tour agency is a web application for people who are going on vacation.
## Used technologies:
- REST API 
- Django 
- React 
- Docker 

## Installation
Change .env-example to .env and after this use [docker-compose](https://docs.docker.com/compose/) to install tour agency.

```bash
docker-compose up -d
```
